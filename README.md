**PROCESAMIENTO Y LIMPIEZA DE DATOS**

Taller desarrollado en Python utilizando NumPy y Pandas para generar, analizar y limpiar un dataset sintético almacenado en archivos CSV. El programa permite generar datos sintéticos, calcular estadísticas descriptivas, detectar y eliminar valores atípicos, y generar automáticamente un dataset limpio junto con un reporte de resultados.

Archivos:
- dataset.csv
- dataset_limpio.csv
- reporte.txt
- Codigo main.py
